Exploring generative music through Conway's Game of Life

Overview

We introduce the cellular automata music generator for Lab 5. This uses Moore's Game of Life in the grid, applying certain rules on how to "birth" a new alive cell or kill a cell. The grid directly ties to a mapping for notes — each row corresponds to a certain pitch in the selected scale chosen by the user.

How It Works

The sound generator uses the basic Web Audio API oscillators. These are then turned into frequencies. I have also added waveform options (sine, triangle, sawtooth, square). I was debating on making certain parts of the map use different types of waveforms, but it sounded quite weird, so I kept it uniform.

Example flow: If you're in A pentatonic (A-C-D-E-G) with 5 rows, and column 3 has cells alive in rows 0 and 2, it plays A and D simultaneously when the playhead reaches that column.

Development Process

My front-end skills are very weak, so for this lab I depended on Copilot querying to build out the front end. There were two main issues that I ran into:

  • Sound quality: The sounds never sounded succinct and did not "flow" with one another. Getting smooth transitions between notes was a challenge.
  • Styling limitations: A typical stylistic issue when it comes to prompting is that it always looks the same. I tried changing it but it would not deviate off its normal style.
